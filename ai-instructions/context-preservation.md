As part of any changes, store a summary of the changes in a markdown file under `docs/what-was-vibed/` folder. 
Ensure the file name is descriptive of the change and/or the motivation for the change.
The file name must begin with a 5 digit date in the format YYYYMMDD, and then a 3 digit counter (note: no dashes or underscores).
Example of the file name: 20260518003-improved-creep-pathing.md.
In a multi turn conversation, you may edit previous files.

If you need historical context, you can refer to the `what-was-vibed` folder for previous changes.
